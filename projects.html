OnBoardPi

OnBoardPi turns a Raspberry Pi or similar single board computer into a data (web) server for you car. The Pi connects to your OBDII port (for reading diagnostic data) and relays things like RPM, Speed, and temperatures via a websocket to any device who has a browser. It is made with NodeJS and Python on the backend and Angular for the web client. This is a passion project which I plan to grow into the future to have more capabilities.

TaffyTangle

A match three game made in Python for a first year assignment. It utilizes the modules 'stddraw', 'stdio', and 'color' from the Princeton Education Booksite. It includes two difficulties, and a hint button for when you are stuck. The difficulties each have a different number of game pieces and score targets. There is a move limit so if you do not reach the target score you lose.

Object Serialization

From a past assignment where I learned about reflective programming in Java, sockets and JSON serialization. The module takes user input and creates objects at runtime with custom fields and values. The server is then tasked with serializing the objects into a JSON string which are then streamed over a socket to some client. Lastly, the client upon receiving the serialized data formats the objects in a pretty form for inspection.
